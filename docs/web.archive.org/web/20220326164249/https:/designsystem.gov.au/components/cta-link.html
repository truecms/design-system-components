Call to action

v2.2.1

Call to action links are visually distinct instructions to users designed to provoke an immediate response using verbs such as 'call now' or 'find out more'.

Released

History
View changes
Install
npm i @gov.au/cta-link
Tags
  • Interactive
  • Navigation
Requires
  • Core
Contributors
  • Overview
  • Rationale
  • Accessibility
  • Code
  • Live demo

Default

A simple and clear link to direct users to a preferred action.

Example

<!--
  Light:  <a class="au-cta-link" href="#">
  Dark:   <a class="au-cta-link au-cta-link--dark" href="#">
-->

<a class="au-cta-link" href="#">Sign up</a>

/*
  Light:  <AUctaLink>
  Dark:   <AUctaLink dark>
*/

import AUctaLink from '@gov.au/cta-link';

<AUctaLink link="/signup" text="Sign up" />
